| property | css                            |
| -------- | ------------------------------ |
| m        | margin                         |
| mt       | margin-top                     |
| mr       | margin-right                   |
| mb       | margin-bottom                  |
| ml       | margin-left                    |
| mx       | margin-left and margin-right   |
| my       | margin-top and margin-bottom   |
| p        | padding                        |
| pt       | padding-top                    |
| pr       | padding-right                  |
| pb       | padding-bottom                 |
| pl       | padding-left                   |
| px       | padding-left and padding-right |
| py       | padding-top and padding-bottom |

</div>
